# moi-signer

This is a sub-package of [moi.js](https://github.com/sarvalabs/moi.js).

The **moi-signer** package represents an MOI account with the ability to sign interactions and messages for cryptographic proof. It provides the necessary tools to sign interactions securely and authenticate interactions on the MOI network.

## Installation
Install the latest [release](https://github.com/sarvalabs/moi.js/releases) using the following command.

```sh
npm install moi-signer
```

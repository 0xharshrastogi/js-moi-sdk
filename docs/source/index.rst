What is moi.js?
----------------
The moi.js is a feature-rich library designed to seamlessly interact with 
the MOI Blockchain and its extensive ecosystem.  It strives to be a compact and 
comprehensive solution, providing developers with all the necessary tools they 
need to unlock the full potential of the MOI Blockchain.
